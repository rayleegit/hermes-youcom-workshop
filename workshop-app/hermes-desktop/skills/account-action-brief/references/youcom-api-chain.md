# You.com API Chain

1. **Search** — discover sources (`results.web`, `results.news`)
2. **Filter** — relevance, freshness, trust
3. **Contents** — read 1–3 URLs as markdown
4. **Research** — multi-source synthesis with citations
5. **Structured Research** (optional) — `output_schema` for predictable fields
6. **Finance Research** (optional) — public-company financial context

## MCP tools (Hermes)
- you-search, you-contents, you-research
- Server: https://api.you.com/mcp

## Governance levers
- include_domains / exclude_domains / boost_domains
- freshness, research_effort (lite → exhaustive)
