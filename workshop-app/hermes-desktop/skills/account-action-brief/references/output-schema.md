# Account Action Brief — Output Schema

## 1. Snapshot
- Company, website, industry, size/scale, business model
- Source confidence rating

## 2. Current Signals
- Recent news, product/strategy updates, leadership changes
- Funding, market, partnership signals

## 3. Why This Account Might Care
- Priorities, pain points, buying triggers, transformation themes
- May include assumptions — label them

## 4. Suggested Actions
- Recommended next step, owner, outreach angle, open questions

## 5. Claims And Sources
- Claim → Source URL → Date/freshness → Confidence

## 6. Review Notes
- Weak claims, missing sources, external-use warnings
