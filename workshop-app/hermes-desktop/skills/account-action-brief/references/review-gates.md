# Review Gates

| State | Meaning |
|-------|---------|
| Draft | Output exists; no human review yet |
| Needs edits | Unsupported claims, weak sources, tone issues |
| Approved for internal use | Safe for meeting prep, internal discussion |
| Approved for external use | Safe for outreach, customer comms |

## Review checklist
- Are all factual claims source-backed?
- Are sources recent enough?
- Are recommendations separated from facts?
- Is internal-only context clearly marked?
- Any claims that must not be used externally?
- Is the suggested next action reasonable?

Default every run to **Draft**.
