# Community Pulse

## When to Use

Facilitators run this weekly to message a workshop alumni community with a **sourced** update and one `/account-action-brief` exercise.

**Delivery:** Copy output → post to Slack/Discord. Do not auto-post.

## Procedure

1. Confirm topic and audience (e.g. "workshop alumni building account briefs").
2. Use You.com Search and Research for 3–5 current signals on the topic.
3. Suggest one company for members to run `/account-action-brief` this week.
4. End with CTA: run your agent, share one learning, ask one governance question.
5. Set review status: **Draft** — facilitator sends manually.

## Output Format

```markdown
## Community Pulse — [DATE]
**Topic:** [TOPIC]

### Signals (sourced)
- [bullet + URL]

### This week's exercise
Run `/account-action-brief` for **[COMPANY]** because **[REASON]**.

### CTA
- Run your agent
- Reply with: company tried, one trusted source, one governance question
```

## Pitfalls

- Do not hype — mark uncertain claims.
- Do not auto-post to Slack, email, or CRM.
- Keep pulses short — members finish in 5 minutes of reading.

## Verification

- [ ] Every signal has a URL
- [ ] One clear exercise company named
- [ ] Draft status noted
