# Delegated procedure — account-action-brief

When `/meeting-prep` identifies a company, run the same evidence chain as `/account-action-brief`:

1. **You.com Search** — recent strategic signals for the company
2. **Select 1–3 URLs** — high-value sources from search results
3. **You.com Contents** — read selected pages as markdown
4. **You.com Research** — synthesize when multi-source context is needed
5. **Format** — six sections in order:
   - Snapshot
   - Current Signals
   - Why This Account Might Care
   - Suggested Actions
   - Claims And Sources
   - Review Notes
6. **Governance** — Draft status; separate facts from recommendations; label assumptions

If the `account-action-brief` skill is installed, follow its SKILL.md and references.
Otherwise load from `account-action-brief/references/output-schema.md` in the same skill pack.

Do not shorten the brief for calendar runs — meeting prep deserves the same source quality.
