# Luma Flyer Prompts

Use these prompts to regenerate or refine the Luma cover images.

## Workshop 1 Cover

Use case: ads-marketing

Asset type: Luma event flyer / cover image, 16:9 horizontal banner

Primary request: Create a polished event flyer for a technical workshop.

Style/medium: high-end editorial tech event poster, crisp digital design, subtle depth, no logos unless represented as plain text, modern enterprise AI aesthetic.

Composition/framing: centered title block with abstract workflow lines, source cards, governance shield motif, and web intelligence nodes; generous margins for Luma cropping.

Color palette: deep charcoal, clean white, electric blue accents, small warm amber accent; balanced, not monochrome.

Text verbatim:
- Hermes + You.com
- Governed Account-Brief Workflow
- [DATE TBD] | [TIME TBD] PT

Constraints: text must be clean, readable, spelled exactly, professional, no extra words, no fake UI brand marks, no QR code, no watermark.

Avoid: illegible text, crowded layout, dark blurry stock-photo background, cartoon style, excessive gradients.
